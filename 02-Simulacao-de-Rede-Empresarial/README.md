# Projeto 02 - Simulação de Rede Empresarial (Cisco Packet Tracer)

## Objetivo

Simular uma infraestrutura de rede empresarial utilizando o Cisco Packet Tracer, conectando diferentes setores da empresa e realizando testes de conectividade.

## Descrição

Este projeto foi desenvolvido durante a disciplina de Redes de Computadores utilizando o Cisco Packet Tracer.

A atividade consistiu na criação de uma rede empresarial simulada, conectando diferentes dispositivos e setores por meio de switches e roteadores. O objetivo foi compreender a estrutura de uma rede corporativa e praticar a configuração e a comunicação entre os equipamentos.

## Equipamentos utilizados

- Cisco Packet Tracer
- Roteador
- Switches
- Computadores
- Smartphones
- Cabos de rede

## Atividades realizadas

- Planejamento da topologia da rede;
- Interligação dos dispositivos;
- Configuração básica dos equipamentos;
- Testes de conectividade entre os dispositivos;
- Simulação de um ambiente corporativo.

## Competências desenvolvidas

- Cisco Packet Tracer
- Topologias de rede
- Infraestrutura de redes
- Configuração básica de dispositivos
- Testes de conectividade
